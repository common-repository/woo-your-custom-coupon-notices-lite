Show Custom Notices when customer applies a coupon.

== Description ==
Woo Your Custom Coupon Notices allows you to show a custom notice to the customer when they apply a coupon during cart / checkout process.
Use more informative messages than "Coupon does not exist!" and many other default messages. 

You can use placeholders/tokens, which will be substituted with the matching value.

This Lite version allows to customize up to three coupons.

You can test/get the full version at http://www.w34all.de.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/woo_your_custom_coupon_notices_lite` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the \'Plugins\' screen in WordPress
3. Customize coupons via the tab \'Messages\" at every coupon.
